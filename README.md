# Project P (temporary name)

Hi, this is the Project P repository.  
Project P is an upcoming metroidvania game. You play as a ghost who fights his way thanks to his umbrella.  
Your umbrella can help you do many things, like floating in the air or on the water, climbing walls, protecting from projectiles, and more !

You will need Godot >=4.5 to run the project :

- Download at https://godotengine.org/download/
- Open the project
- Install the export templates needed to run the game on your OS
- Export the project
- Enjoy !
